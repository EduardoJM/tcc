associado ao Problema de Valor de Contorno que modela a deflexĂŁo de uma viga apoiada sobre seus extremos. A \index[remissivo]{FunĂ§ĂŁo Triangular}funĂ§ĂŁo triangular trabalhada ĂŠ definida como
\[ \phi_i (x) = \begin{cases} 0 & \text{se } 0 \leqslant x \leqslant x_{i - 1}\\[5pt] \dfrac{x-x_{i-1}}{h_{i-1}} & \text{se } x_{i-1} < x \leqslant x_i\\[10pt] \dfrac{x_{i+1}-x}{h_i} & \text{se } x_i < x \leqslant x_{i+1}\\[5pt] 0 & \text{se } x_{i+1}<x\leqslant 1 \end{cases} \text{,} \]
onde \(h_i = x_{i+1}-x_{i}\) e \(h_{i-1}=x_i-x_{i-1}\). A Figura 3.2 apresenta uma ilustraĂ§ĂŁo da funĂ§ĂŁo \(\phi_i(x)\) definida. A funĂ§ĂŁo exata, \(y(x)\), que torna um funcional estacionĂĄrio, ĂŠ aproximado, entĂŁo, por meio da funĂ§ĂŁo aproximadora
\[ v(x)=\sum_{i=1}^{n} a_i \phi_i(x) \text{.} \]

Observe que, de acordo com a definiĂ§ĂŁo das funĂ§Ăľes de forma \(\phi_i(x)\), a funĂ§ĂŁo \(v(x)\), definida como combinaĂ§ĂŁo linear das \(\phi_i(x)\), funcionarĂĄ apenas para funcionais que dependem apenas de derivadas de primeira ordem, visto que a derivada de segunda ordem de qualquer \(\phi_i(x)\) ĂŠ nula.

A primeira derivada das funĂ§Ăľes \(\phi_i(x)\) em relaĂ§ĂŁo a \(x\) ĂŠ escrita como

\[ \phi_i ' (x) = \begin{cases} 0 & \text{se } 0 \leqslant x < x_{i - 1}\
\[5pt] \dfrac{1}{h_{i-1}} & \text{se } x_{i-1} < x < x_i\
\[10pt] \dfrac{-1}{h_i} & \text{se } x_i < x < x_{i+1}\
\[5pt] 0 & \text{se } x_{i+1}<x\leqslant 1 \end{cases} \text{,} \]
de onde a derivada de \(v(x)\) em relaĂ§ĂŁo a \(x\) ĂŠ
\[ v'(x)=\sum_{i=1}^{n} a_i \phi_i '(x) \text{.} \]

Aproximando \(y(x)\) por meio da funĂ§ĂŁo \(v(x)\) definida em , o funcional ĂŠ escrito como

\[ I = \int_{0}^{1} \left [ p(x) \left ( \sum_{i=1}^{n} a_i \phi_i '(x) \right )^2 + q(x) \left ( \sum_{i=1}^{n} a_i \phi_i (x) \right )^2 - 2f(x) \left ( \sum_{i=1}^{n} a_i \phi_i (x) \right ) \right ] dx \text{.} \]

Para determinar os coeficientes \(a_j\), as derivadas parciais de em relaĂ§ĂŁo aos \(a_j\) devem ser todas nulas, ou seja,

\[ \frac{\partial I}{\partial a_j} = 0 \text{,} \]
onde \(j=1\), \(2\), \(\dots\), \(n\). Utilizando A Regra de Leibniz (Teorema \ref{teorema:regra_de_leibniz} do ApĂŞndice \ref{apend:regra_de_leibniz}),
\[ \frac{\partial I}{\partial a_j} = \int_{0}^{1} \left [ 2p(x) \left ( \sum_{i=1}^{n} a_i \phi_i'(x) \right ) \phi_j'(x) + 2q(x) \left ( \sum_{i=1}^{n} a_i \phi_i(x) \right ) \phi_j(x) - 2f(x)\phi_j(x) \right ] dx \text{.} \]

Em foi utilizado o fato de que a derivada de \(\displaystyle \sum_{i=1}^{n} a_i \phi_i(x)\) em relaĂ§ĂŁo a algum \(a_j\), onde \(1\leqslant j\leqslant n\), resulta em \(\phi_j(x)\). O mesmo vale para a derivada de \(\displaystyle \sum_{i=1}^{n} a_i \phi_i'(x)\) em relaĂ§ĂŁo a algum \(a_j\), \(1\leqslant j\leqslant n\), que resulta em \(\phi_j'(x)\).

Aplicando a distributividade na primeira e segunda parcela do integrando em , tem-se

\[ \frac{\partial I}{\partial a_j} = \int_0^1 \left [ \sum_{i=1}^n \left ( 2p(x)a_i \phi_i'(x)\phi_j'(x) \right) + \sum_{i=1}^n \left ( 2q(x)a_i\phi_i(x)\phi_j(x) \right) - 2f(x)\phi_j(x) \right ] dx \]
\[ \frac{\partial I}{\partial a_j} = \int_0^1 \left [ \sum_{i=1}^n \left ( 2p(x)a_i \phi_i'(x)\phi_j'(x) + 2q(x)a_i\phi_i(x)\phi_j(x) \right ) \right ] dx - \int_0^1 2f(x)\phi_j(x)dx \text{,} \]
de onde, alterando a ordem de integraĂ§ĂŁo e soma, resulta em
\[ \frac{\partial I}{\partial a_j} = \sum_{i=1}^n \left ( \int_0^1 \left [ 2p(x)a_i\phi_i'(x)\phi_j'(x) + 2q(x)a_i\phi_i(x)\phi_j(x) \right ] dx \right ) - \int_0^1 2f(x)\phi_j(x)dx \text{.} \]

De , como \(a_i\) ĂŠ constante na integraĂ§ĂŁo em relaĂ§ĂŁo a \(x\), tem-se

\[ \frac{\partial I}{\partial a_j} = \sum_{i=1}^{n} \left ( a_i \int_{0}^{1} \left [ 2p(x)\phi_i'(x)\phi_j'(x) + 2q(x)\phi_i(x)\phi_j(x) \right ] \right ) - \int_{0}^{1} 2f(x)\phi_j(x)dx \text{,} \]
que, aplicando a condiĂ§ĂŁo \(\dfrac{\partial I}{\partial a_j}=0\), obtĂŞm-se
\[ \sum_{i=1}^{n} \left ( a_i \int_{0}^{1} \left [ 2p(x)\phi_i'(x)\phi_j'(x) + 2q(x)\phi_i(x)\phi_j(x) \right ] \right ) - \int_{0}^{1} 2f(x)\phi_j(x)dx = 0 \text{,} \]
resultando em um sistema de \(n\) equaĂ§Ăľes com \(n\) incĂłgnitas da forma
\[ \sum_{i=1}^{n} \left ( a_i \int_{0}^{1} \left [ 2p(x)\phi_i'(x)\phi_j'(x) + 2q(x)\phi_i(x)\phi_j(x) \right ] dx \right ) = \int_{0}^{1} 2f(x)\phi_j(x)dx \text{,} \]
onde \(j=1\), \(2\), \(\dots\), \(n\).

Chamando

\[ m_{j,i}=\int_{0}^{1} \left [ 2p(x)\phi_i'(x)\phi_j'(x) + 2q(x)\phi_i(x)\phi_j(x) \right ]dx \text{,} \]
e
\[ b_j=\int_{0}^{1} 2f(x)\phi_j(x)dx \text{,} \]
o sistema pode ser escrito na forma
\[ \displaystyle\sum_{i=1}^n a_i\cdot m_{j,i}=b_j \text{,} \]
onde \(j=1,2,\dots,n\). Seja as matrizes \(M=[m_{j,i}]\) de ordem \(n\times n\), \(B=[b_j]\) de ordem \(n\times 1\) e \(A=[a_j]\) de ordem \(n\times 1\), entĂŁo o sistema ĂŠ escrito como \(M\cdot A=B\). A matriz \(M\) ĂŠ a matriz de coeficientes, a matriz \(A\) ĂŠ a matriz coluna de incĂłgnitas e \(B\) ĂŠ a matriz coluna contendo os termos independentes.

A partir de e , os produtos \(\phi_i(x)\phi_j(x)\) e \(\phi_i'(x)\phi_j'(x)\) sĂŁo nĂŁo nulos apenas quando \(i=j-1\), \(i=j\) e \(i=j+1\). Consequentemente, \(m_{j,i}\) ĂŠ nĂŁo nulo apenas para \(i=j-1\), \(i=j\) e \(i=j+1\).
